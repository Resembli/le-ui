title: Fixed Size Examples
---

Remember to import the `<ListHorizontal />` component. It is not shown in the examples below.

```js
import { ListHorizontal } from "@resembli/react-virtualized-window"
```

For more on the `<ListHorizontal />` component, please refer to the component's [API documentation](/docs/react-virtualized-window/list-horizontal)

## Basic Example

```jsx live
function FixedSizeHorizontalList() {
  const catImages = [
    ["/cat-images/cat-1.jpg", "Example image of cat 1"],
    ["/cat-images/cat-2.jpg", "Example image of cat 2"],
    ["/cat-images/cat-3.jpg", "Example image of cat 3"],
    ["/cat-images/cat-4.jpg", "Example image of cat 4"],
    ["/cat-images/cat-5.jpg", "Example image of cat 5"],
    ["/cat-images/cat-6.jpg", "Example image of cat 6"],
  ]

  const data = Array(1000)
    .fill(0)
    .map((_, i) => {
      const index = i % catImages.length
      const props = {
        src: catImages[index][0],
        alt: catImages[index][1],
      }
      return props
    })

  return (
    <div style={{ height: 250, border: "1px solid grey" }}>
      <ListHorizontal data={data} defaultColumnWidth={200}>
        {(props, styles) => {
          return (
            <div
              className="small-font-for-mobile"
              style={{
                padding: 10,
                ...styles,
              }}
            >
              <img src={props.src} alt={props.alt} style={{ width: "100%", height: "100%" }} />
            </div>
          )
        }}
      </ListHorizontal>
    </div>
  )
}
```
